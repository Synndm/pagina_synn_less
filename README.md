# 🌐 Página Synn (LESS)

## 📌 Sobre o projeto
Este projeto é uma página web desenvolvida utilizando **HTML, CSS e LESS**, com foco em organização de estilos e boas práticas no desenvolvimento front-end.

O objetivo foi praticar a utilização do pré-processador LESS para tornar o CSS mais modular, reutilizável e escalável.

---

## 🚀 Tecnologias utilizadas
- HTML5  
- CSS3  
- LESS    

---

## 💻 Funcionalidades
- Layout responsivo  
- Estrutura organizada com LESS  
- Estilização modular  
- Compilação automática de estilos com LESS
